# Pi Global Instructions

- Keep responses concise and factual.
- Prefer small, focused changes over broad rewrites.
- Do not commit secrets, API keys, or credentials.
- Do not run destructive commands unless explicitly requested.
- Prefer `bd` (beads) for task tracking in projects that use it; do not create markdown TODO lists.

## Search

For repository file and content searches, use the FFF-backed search tools whenever available.
Prefer `fffind` for file/path discovery and `ffgrep` for content search.
Do not use native `find` or `grep` when an equivalent FFF tool is available.
