# Debug

Use this skill when diagnosing an error, failing command, or unexpected behavior.

- Reproduce the issue when feasible.
- Identify the smallest failing surface.
- State the likely root cause before changing code.
- Prefer one focused fix and verify it.
- If blocked by missing environment or permissions, explain exactly what output is needed.
